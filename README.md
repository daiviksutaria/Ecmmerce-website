This E-commerce Website is a responsive web application designed to simulate an online shopping platform. The project demonstrates core web development concepts including dynamic product listing, shopping cart functionality, and interactive user interface design. It provides users with a smooth browsing experience and basic e-commerce features such as viewing products, adding items to the cart, and managing selected products.

The application focuses on clean UI design, structured code organization, and user-friendly navigation. It is built to showcase practical implementation of front-end development skills and fundamental business logic used in online retail platforms.

Key Features

Product catalog with detailed view

Add to cart functionality

Cart management system

Responsive design for different screen sizes

Clean and interactive user interface

Technologies Used

HTML

CSS

JavaScript
